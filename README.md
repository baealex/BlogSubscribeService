## fallinto

블로그 팔로워 시스템

---

들어가야 할 내용

- 회원가입
  - 이메일 인증
  - 비밀번호 / 확인
- 최초 로그인시 구독메뉴
  - 카테고리 선정
  - 구독 버튼
- 메인
  - 자신이 구독한 카테고리
  - 해당 카테고리 구독한 사람이 몇명인지
  - 일주일 이내에 작성된 글 보여주기
  - 해당 카테고리 인기글 3개 보여주기
    - 디스커스의 하트 or 따로 만들어야 할 듯
- 정보
  - 이메일로 소식 받기(인증된 경우) 선택/해제
  - 구독 카테고리 변경
  - 비밀번호 변경
  - 회원탈퇴
    - 재확인
